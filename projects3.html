Grimsby & Clee Sells is a dynamic shop program that I developed using ASP.NET Core, incorporating SQL, HTML, JavaScript, and C#. This platform allows users to buy and sell products, as well as log in, manage their accounts, and update personal details with ease. A key focus of the project was robust error handling, ensuring that new users never encountered unexpected errors, creating a seamless and professional user experience. The system was designed with security and efficiency in mind, providing a smooth transaction process and intuitive user interface. This project reflects my expertise in full-stack development and my commitment to building reliable, user-friendly applications.
